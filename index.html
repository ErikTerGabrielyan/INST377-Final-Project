Search UMD Courses and Professors

Select either 'Courses' or 'Professors' from the dropdown to enable searches for whichever one you wish.
If searching for courses, search for the course ID. For example: MATH140, INST377
If searching for a professor, make sure to search for the professor's full first and last name, with spaces in between. For example: Erik Ter-Gabrielyan
